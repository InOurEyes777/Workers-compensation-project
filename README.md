# **Система прогнозирования стоимости страховых выплат**

Данный проект представляет собой комплексное решение для анализа данных о страховых случаях и прогнозирования итоговой стоимости выплат (UltimateIncurredClaimCost). Проект включает в себя полный цикл машинного обучения: от разведочного анализа данных (EDA) до построения интерпретируемых моделей и развертывания в виде интерактивного веб-приложения.

## Основные решенные задачи:

### **1. Детальный анализ данных (EDA):**
- Исследование временной динамики выплат по годам и месяцам.
- Анализ распределения целевой переменной и проверка на нормальность
- Выявление и обработка выбросов методами Тьюки и Z-оценки.
- Корреляционный анализ и исследование зависимостей между признаками.
- Анализ категориальных переменных и текстовых описаний травм.

### **2. Инженерия признаков:**
- Генерация новых признаков на основе дат (месяц, день недели, задержка отчета).
- Создание интерактивных признаков (взаимодействие возраста и зарплаты).
- Векторизация текстовых описаний травм методом TF-IDF.

### **3. Моделирование и оптимизация:**
- Обучение базовых моделей (Linear Regression, Ridge) и ансамблевых методов (Random Forest, XGBoost, CatBoost).
- Автоматический подбор гиперпараметров с помощью библиотеки Optuna с использованием кросс-валидации.
- Оценка качества моделей 

### **4. Интерпретируемость:**
- Анализ важности признаков с помощью встроенных методов моделей.
- Использование библиотеки SHAP для детальной интерпретации предсказаний (Summary Plot, Waterfall Plot).
  
### **5. Разработка приложения:**
- Создание многостраничного веб-приложения на Streamlit с удобным интерфейсом для анализа и предсказания.

## **Установка и запуск**

### **1\. Клонируйте репозиторий:**

"git clone <ссылка>"

### **2\. Перейдите в директорию проекта:**

&#x09;cd Workers-compensation-project

### **3\. Установите зависимсоти:**
&#x09;pip install -r requirements.txt

4\. Запустите приложение:

&#x09;streamlit run app.py
